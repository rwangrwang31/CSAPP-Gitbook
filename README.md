# 深入理解计算机系统 (CSAPP) 电子书

欢迎阅读《深入理解计算机系统》(Computer Systems: A Programmer's Perspective) 中文版电子书。

本书基于原书第3版，包含完整的12章内容以及附录，涵盖了计算机系统的核心概念：

## 📚 内容概览

### 第一部分：程序结构和执行
- 第1章：计算机系统漫游
- 第2章：信息的表示和处理  
- 第3章：程序的机器级表示
- 第4章：处理器体系结构
- 第5章：优化程序性能
- 第6章：存储器层次结构

### 第二部分：在系统上运行程序
- 第7章：链接
- 第8章：异常控制流
- 第9章：虚拟内存

### 第三部分：程序间的交互和通信
- 第10章：系统级 I/O
- 第11章：网络编程
- 第12章：并发编程

## 🛠️ 技术实现

- 渲染引擎：HonKit（兼容 GitBook）
- 部署平台：Vercel
- 内容来源：CSAPP_OCR_FINAL 原始 OCR 分块内容

## 📖 使用指南

左侧目录提供完整的章节导航，可按顺序阅读或跳转到感兴趣的章节。每章都包含详细的小节索引和配图。
